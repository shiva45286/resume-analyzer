# 🚀 AI Resume Analyzer

An intelligent web application that analyzes resumes against job descriptions and provides ATS scores, missing keywords, and improvement suggestions.

---

## 📌 Features

* ✅ ATS Score Calculation
* 🔍 Keyword Matching
* 📊 Interactive Chart (Chart.js)
* 🤖 Resume Improvement Suggestions
* 💬 Chatbot Assistant
* 🔐 User Login & Registration

---

## 🖥️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python (Flask)
* **Database:** SQLite
* **Libraries:**

  * pdfplumber (PDF parsing)
  * Flask-Login (Authentication)
  * Chart.js (Visualization)

## 📊 How It Works

1. Upload your resume (PDF)
2. Paste job description
3. System compares keywords
4. Generates:

   * ATS Score
   * Missing Keywords
   * Improvement Suggestions
